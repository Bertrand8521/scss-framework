Utilisation

Version SCSS

On peut personnaliser les options du framework en modifiant les options dans scss/_variables.scss, ensuite faire "sass scss/framework.scss css/framework.css" afin de regénérer le fichier CSS, on peut par exemple changer le ratio minimal que la grille utilise (par défaut 12) afin d'avoir un positionement plus précis ou un fichier CSS plus petit.

Version CSS:

Inclure simplement css/framework.css dans les fichiers html
